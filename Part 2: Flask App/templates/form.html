This App predicts any STOCK price

Please enter the following information:

  • Stock Name is the CAPITALIZED letters for the stock. AAPL fpr Apple, BTC-USD fro Bitcoin, TSLA for Tesla.
  • Epochs is the number of passes of the entire training dataset, the machine learning algorithm has completed. More the epochs, better the accuracy. Epoch can be set at 50 for a good performance.
  • Ahead is the number of days you want predict ahead of time. Less the number in "Ahead", higher the accuracy beacasue machine will have hard time predicting further into the future. 
  • Days is the number of days you want to use as the prediction dataset to predict the future. For example, you can use 30 days of data to predict the 31st day (ahead =1) or use the same data to predict 40th day (ahead=10)
